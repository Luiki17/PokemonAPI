# Pokédex API

Este es un proyecto simple de una API REST para consultar y guardar Pokémon usando Spring Boot y H2 como base de datos en memoria. 

La API obtiene datos desde la [PokéAPI](https://pokeapi.co/) y permite almacenarlos localmente.

---

## Tecnologías usadas

- Java 17+
- Spring Boot 3.x
- Spring Data JPA
- H2 Database (en memoria)
- RestTemplate para consumir PokéAPI
- Maven para gestión de dependencias
